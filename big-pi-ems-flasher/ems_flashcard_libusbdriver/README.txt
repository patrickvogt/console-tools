################################################################################
# README - LibUSB driver for EMS GB flash card 64M (unsigned)                  #
################################################################################

These driver(s) are unsigned. You need to boot Windows Vista/Windows 7/Windows 8
 in a mode which won't check for signed drivers. Then you can install these 
 driver(s) with right click on the *.INF file and choose 'Install'
 
After this install the EMS GB flash card 64M should be recognized and the 
 pi-ems-flasher should work (with PyUSB installed)
